[nav 페이지]
1. 로고 클릭 시 홈 페이지로 이동한다.
2. 네비게이션 내 각 버튼 클릭 시 해당 페이지로 이동한다.
3. “제목”, “사람”, “장르” 검색이 가능하며, ENTER 키 또는 검색 버튼 클릭 시 결과 화면으로 전환된다.
4. 검색 결과 화면:
   4-1. “더 다양한 검색어가 필요하다면!” 등의 권장 검색어 메시지를 출력한다.
   4-2. 검색어를 포함하는 콘텐츠와 관련 항목을 그리드 형태로 출력하며, 반응형 레이아웃과 로딩 상태를 고려한다.
   4-3. 모바일 환경에서도 사용이 용이하도록 검색 UI 및 결과 정렬/필터 옵션을 제공한다.

[목록 출력 페이지]
1. 포스터에 마우스 오버 시 썸네일과 간단 정보(제목, 평점 등)가 표시된다.
   - 썸네일 클릭 시 영상 재생 화면으로, 정보 클릭 시 모달로 상세 정보가 출력된다.
2. 각 주제별로 기본 5개 항목을 출력하며, “이전”, “다음” 버튼(또는 슬라이더)으로 전환이 가능하고, 현재 페이지 번호를 상단에 표시한다.
3. 순위나 통계 기반 정렬에 따른 콘텐츠 목록을 제공하며, 데이터가 없을 경우 대체 콘텐츠나 “로딩 중” 메시지를 표시한다.

[home 페이지]
1. 메인 배너 영상은 50초간 자동 재생 후 썸네일 이미지로 전환된다.
2. 배너 내 재생 버튼 클릭 시 해당 영상 재생 화면으로 전환된다.
3. 상세정보 버튼 클릭 시 모달 창으로 해당 영상의 상세 정보를 출력한다.
4. 메인 영역에 페이지 분석에 명시된 40개의 콘텐츠(타이틀 및 관련 정보를 포함한 카드 형태)를 출력하며, 반응형 그리드 및 페이징(또는 무한 스크롤) 기능을 지원한다.
5. 포스터에 마우스 오버 시 썸네일과 간단 정보가 출력되며, 썸네일 클릭 시 영상 재생, 정보 클릭 시 모달로 전환된다.
6. 각 주제별로 5개 항목을 출력하며, 전환 버튼을 통해 목록을 변경할 수 있고 현재 페이지 번호를 상단에 표시한다.
7. 순위나 통계 데이터를 기반으로 콘텐츠가 출력된다.
8. “시청중인 콘텐츠” 섹션에서는 사용자가 최근에 시청한 영상부터 시간 순으로 출력하며, 이어보기 기능을 지원한다.

[시리즈 페이지]
1. 전체 레이아웃 및 UI 구성은 홈 페이지와 동일하게 유지한다.
2. 장르 버튼 클릭 시, 페이지 분석에 명시된 시리즈 전용 전체 장르 목록을 출력하며, 필터링 및 정렬 옵션을 제공한다.
3. “목록”과 “추천 콘텐츠” 버튼을 제공하며, 추천 콘텐츠 버튼은 별도의 알고리즘(예: 유사 장르, 시청 기록 등)에 따라 작동한다.
4. 각 버튼 클릭 시 전체 시리즈 콘텐츠 목록을 대상으로 해당 항목들을 반응형 그리드 또는 슬라이더 형태로 출력한다.
5. 페이지 전환 및 로딩 상태, 에러 처리 등 접근성과 반응형 디자인을 고려한다.

[영화 페이지]
1. 전체 레이아웃 및 UI 구성은 홈 페이지와 동일하게 유지한다.
2. 장르 버튼 클릭 시, 페이지 분석에 명시된 영화 전용 전체 장르 목록을 출력하며, 필터 및 정렬 기능을 제공한다.
3. “목록”과 “추천 콘텐츠” 버튼을 제공하며, 추천 콘텐츠 버튼은 앞서 정의한 방식으로 작동한다.
4. 각 버튼 클릭 시 전체 영화 콘텐츠 목록을 대상으로 해당 항목들을 반응형 그리드 또는 슬라이더 형태로 출력한다.
5. 시리즈 페이지와의 차별점:
   - 영화 페이지는 영화에 특화된 장르와 분류 기준을 적용한다.
   - 출력 내용(정보 구성, 카드 디자인 등)이 영화 콘텐츠에 최적화되어 있다.
6. 접근성 및 반응형 디자인, 로딩/페이징 기능을 추가로 고려한다.

[new! 요즘 대세 콘텐츠 페이지]
1. TOP10 시리즈와 TOP10 영화는 순위와 포스터가 하나의 카드 형태로 출력된다.
2. 각 카드에는 기존 출력 방식(포스터, 제목, 간략 정보 등)과 함께 순위 번호가 명시되며, 터치 및 클릭 인터랙션을 지원한다.
3. 카드 레이아웃은 반응형 디자인으로 다양한 해상도에서 최적의 표시를 보장한다.

[내가 찜한 리스트 페이지]
1. 사용자가 찜(즐겨찾기)한 모든 콘텐츠를 목록(또는 그리드) 형태로 출력한다.
2. 찜한 콘텐츠는 사용자 계정에 저장되며, 수정 및 삭제 기능(찜 해제)이 포함된다.
3. 반응형 UI와 페이징(또는 무한 스크롤) 기능을 지원한다.

[언어별로 찾아보기 페이지]
1. 페이지 접속 시 전체 콘텐츠 목록이 기본으로 출력된다.
2. “원어” 콘텐츠를 별도로 구분하여, 해당 방식(자막/더빙 등)을 지원하는 콘텐츠만 출력할 수 있도록 옵션을 제공한다.
3. 언어 선택 옵션은 데이터에 포함된 언어 종류를 기준으로 드롭다운 혹은 버튼 형태로 제공된다.
4. 정렬 기준(예: 최신순, 인기순 등)을 선택하여 콘텐츠를 정렬할 수 있다.
5. 다중 설정(언어, 정렬 등)이 중복 적용되어 전체 콘텐츠가 필터링되며, 필터 초기화 및 재설정 기능을 포함한다.
6. 로딩 상태, 에러 처리 및 접근성(예: 키보드 네비게이션) 기능을 고려한다.
