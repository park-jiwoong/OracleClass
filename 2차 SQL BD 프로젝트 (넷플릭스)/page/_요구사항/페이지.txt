- 검색
 - 더 다양한 검색어가 필요하시다면!: 
 - 결과 출력

1. 로고를 누르면 홈 페이지로 이동한다.
2. 해당 버튼을 클릭하면 해당 페이지로 이동한다. 
3. 제목, 사람, 장르로 검색이 가능하며 ENTER 혹은 검색을 누르면 결과가 출력된다.
4. 결과
 4-1. 더 다양한 검색어가 필요하다면! 으로 권장 내용을 출력한다.
 4-2. 검색 글자를 포함하는 내용과 관련된 내용을 그리드 형태로 출력한다.

홈 페이지
- 상단 배너 출력
  - 이름 / 재생여부 / 상세정보 / 음소거 / 연령 등급
- TOP10의 경우 랭크 / 썸네일 을 한개로 출력
- 페이지 분석에 존재하는 메인 페이지 내용 출력
- 각 타이틀 / 카드들로 출력
- FOOTER 출력

1. 배너의 영상은 50초 재생 후 썸네일로 전환된다.
2. 재생 버튼을 클릭하면 영상 재생으로 이동한다. 
3. 상세정보를 클릭하면 해당 영상의 모달을 출력한다.
4. 페이지 분석의 메인 페이지에 존재하는 40개를 TITLE로 관련 내용을 출력한다. 
5. 포스터를 호버하는 경우 썸네일과 간단 정보가 출력되며 썸네일은 영상재생, 정보는 모달이 출력된다.
6. 각 주제에 대해 5개씩 출력되며 버튼을 통해 전환이 가능하고 목록 상단에 몇번째인지 출력한다.
7. 각 통계에 대한 순위로 출력된다
8. 시청중인 콘텐츠는 가장 최근에 본 영상부터 출력된다.

- 기본 내용은 홈 페이지와 동일하다.
- 영화 / 장르 버튼이 존재한다.
- 추천 콘텐츠 버튼 => 추천 / 출시일 / 오름차순 / 내림차순

1. 출력되는 틀은 홈 페이지와 동일하다.
2. 장르 버튼을 클릭하면 전체 장르가 출력되며 이 내용은 페이지 분석의 영화 페이지에 명시한 내용을 출력한다.
3. 목록과 추천 콘텐츠 버튼이 존재하며 추천 콘텐츠 버튼에는 위 정의한 내용의 버튼이 존재한다.
4. 각 버튼에 대해 전체 목록을 대상으로 출력한다.
5. 시리즈 페이지와의 차별점은 서로 장르에 대한 출력 내용이 다르고 전체 내용에 대해 해당 내용으로 분류할 수 있다.

- 오늘 대한민국의 TOP10 시리즈
- 넷플릭스의 새로운 콘텐츠
- 기다림이 아깝지 않은 콘텐츠
- 이번 주 공개 콘텐츠
- 오늘 대한민국의 TOP10 영화
- 다음 주 공개 콘텐츠
- FOOTER

1. TOP10 시리즈, TOP10 영화는 목록에 출력되는 내용이 순위, 포스터를 한개의 카드로 출력된다.
2. 각 카드에 출력되는 내용은 기존 출력 방식과 동일하다.

- 선호하는 설정을 선택한다
 - 원어 / 더빙 / 자막
 - 언어 선택
 - 정렬 기준
- 목록 출력

1. 접속 시 전체 목록을 출력한다.
2. 원어는 해당 방식을 지원하는 내용에 포함되는 내용 출력
3. 언어 선택은 데이터의 언어 종류에 포함되는 내용을 선택
4. 정렬 기준을 선택하여 출력
5. 설정은 중복으로 적용되며 포함되는 전체 내용을 출력한다.
