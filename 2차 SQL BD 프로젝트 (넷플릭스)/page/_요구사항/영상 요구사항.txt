1. 시청한 콘텐츠의 시청 종료 시점을 기록한다.
2. 멤버십에 따라 영상 중간 광고 여부를 결정한다.
3. 해당 버튼들의 기능은 동일하게 작동한다.
4. 일시정지 후 10초가 지나면 시간경과 후 에 대한 정보가 출력된다.
5. 영상 종료 후 크래딧 영상 보기 / 다음화 중 선택하여 넘어갈 수 있으며 상단 화살표를 선택하는 경우 배너로 돌아간다. 

1. 모달에서는 전체 에피소드와 시즌을 확인할 수 있어야 한다.
2. 함께 시청된 콘텐츠에는 해당 영상과 비슷한 장르를 모아 출력한다.
3. 영상의 정보를 출력하며 관람등급을 클릭하면 최하단에 위치한 상세정보 위치로 이동한다.
4. 출연에서 이름을 클릭하면 해당 배우가 참여한 영상이 목록형태로 출력된다.
5. 장르, 시리즈 특징은 해당 장르, 시리즈 특징에 포함된 영상 전체를 그리드형태로 출력한다.
6. 재생 버튼은 해당 영상을 시청한 경우와 시청하지 않은 경우로 분류한다.
   6-1. 있는 경우 다음화를 재생하며 없는 경우 1화를 시작 부분부터 재생한다.
7. 최 상단의 영상은 약 40초를 출력하고 썸네일로 전환된다.
8. 시청자 추천 여부도 출력한다.
