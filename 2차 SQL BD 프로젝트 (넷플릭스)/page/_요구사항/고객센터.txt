오류코드
- 오류코드 작성 혹은 메시지 작성 
    - 해결방법 검색
검색 후
- 검색 내용 출력
  - 도움이 될 만한 문서
  - 추가 검색 결과

1. 표시된 오류코드를 검색한다.
2. 해당 오류코드를 상탄에 출력하고 해당하는 문서를 출력한다.
3. 추가검색 결과로 (상세번호)에 포함된 내용을 목록으로 출력한다.
4. 계정과 디바이스 이용 여부 출력 및 내용을 출력한다.
5. 관련 문서를 출력한다.

검색
- 질문, 주제 또는 문제 입력
- 도움이 될만한 문서
- 검색 후 출력 내용은 오류코드와 동일

1. 검색한 내용이 포함된 내용을 전체 출력한다.
2. 검색 결과는 1 - 6 / 2부터 7개씩 목록형태로 출력한다.
3. 상세 정보에서는 관련된 내용은 목록 형태로 출력된다.
4. +버튼을 클릭하는 경우 해당 내용을 출력한다.
