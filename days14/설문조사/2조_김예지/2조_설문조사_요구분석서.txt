설문조사 요구분석서

● 메인 페이지 (최승호)
1. 사용자가 원하는 설문조사 질문내용 출력
2. 사용자가 원하는 질문 항목 출력
3. 투표하기 버튼을 통해 설문자 정보 등록, 참여자수 , 상태 정보 저장
4. 결과보기 버튼을 통한 설문결과 내용 출력

● 설문 목록 (김예지)
1. 설문 홈페이지에 사용자로 가입하려면 사용자아이디, 비밀번호, 이름, 주민등록번호를 입력해야 한다.
2. 사용자는 사용자아이디로 식별한다.
3. 사용자는 설문을 여러 개 작성할 수 있고, 하나의 설문은 한 명의 사용자만 작성할 수 있다.
4. 사용자가 설문을 작성하면 설문에 대한 설문번호, 질문, 시작일, 종료일, 항목, 참여자수, 상태 정보를 유지해야 한다.
5. 사용자는 여러 개의 설문에 참여할 수 있고, 설문 하나에는 여러 명의 사용자가 참여할 수 있다.
6. 사용자가 설문에 참여하면 참여일자와 선택항목 정보를 유지해야 한다.
7. 설문은 설문번호로 식별한다.

● 설문 작성 (최인석)
1. 관리자가 질문 사항을 작성한다.
2. 관리자가 설문 기간을 지정한다.
   1) 시작일을 지정한다.
   2) 종료일을 지정한다.
   3) 시작일과 종료일은 년/월/일 형식을 따른다.
3. 관리자가 항목수를 설정한다.
   1) 최소 2개에서 최대 5개까지로 한다.
4. 관리자가 각 항목의 내용을 지정한다.
5. 확인 버튼을 눌러서 최종적으로 설문 작성을 마친다. 혹은 취소 버튼을 눌러서 작성 중인 설문 내용을 폐기할 수 있다.
